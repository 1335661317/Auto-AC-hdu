# auto-AC-hdu<br>
HDU OJ自动AC机 <br>
看过了网上很多的OJ 自动刷题机，但是很多不能直接用，代码还没有想象中那么简洁，所以自己最终整合出这个Auto AC。
已放出可执行文件哦！<br>
使用方法：<br>
1、下载整个zip包！<br>
2、解压后打开exe文件<br>
3、打开浏览器登录你的hdu账号<br>
4、右键审查里面找cookice信息<br>
5、按照控制台提示信息输入相应信息<br>
6、点击回车，(。・`ω´・)然后，开始自动刷题！<br>
<br>
需要环境：x64系统 VS2015程序运行环境<br><br>
